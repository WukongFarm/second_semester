 🤖 Многофункциональный Telegram-бот (NeuroSmiBot)

Этот проект представляет собой многофункционального Telegram-бота, который объединяет в себе несколько полезных инструментов: от прогноза погоды и конвертации валют до анализа тональности текста и общения с языковой моделью. Бот разработан с использованием современных библиотек Python и следует лучшим практикам разработки, включая использование системы контроля версий и модульную структуру.

## 🌐 Использованные API и их описание

Бот интегрируется со следующими внешними сервисами и API:

1.  **Open-Meteo (для погоды)**
    *   **Описание:** Бесплатный API для получения прогнозов погоды, не требующий API-ключа. Используется для получения данных о температуре, осадках и коде погоды на основе географических координат.
    *   **Endpoint:** `https://api.open-meteo.com/v1/forecast`

2.  **Nominatim (OpenStreetMap) (для геокодинга)**
    *   **Описание:** Сервис для преобразования названия города в географические координаты (широту и долготу), необходимые для запроса в Open-Meteo.
    *   **Endpoint:** `https://nominatim.openstreetmap.org/search`

3.  **Центральный Банк РФ (для курсов валют)**
    *   **Описание:** Официальный XML-документ, публикующий ежедневные курсы валют. Бот парсит этот XML для конвертации между различными валютами, включая рубль.
    *   **Endpoint:** `https://www.cbr.ru/scripts/XML_daily.asp`

4.  **QR Coder (для генерации QR-кодов)**
    *   **Описание:** API для создания QR-кодов из текста или ссылок. Требует ключ `QRCODER_API_KEY`.
    *   **Endpoint:** `https://www.qrcoder.co.uk/api/v4/`

5.  **Gen API (для AI-ассистента)**
    *   **Описание:** Платформа, предоставляющая доступ к различным большим языковым моделям (LLM). В данном боте используется для ответов на вопросы пользователей по команде `/ask`. Требует ключ `GEN_API_KEY`.
    *   **Endpoint:** `https://api.gen-api.ru/api/v1/networks/glm-5`

## 🧠 Базовая модель ИИ

В качестве базовой модели для генерации ответов на вопросы (команда `/ask`) используется **GLM-5**, доступная через API сервиса Gen-API. Эта модель предназначена для ведения диалога, ответов на вопросы и генерации текста на естественном языке.

## 🤖 Модель трансформера и решаемая задача

Для анализа тональности текста (команда `/sentiment`) используется модель-трансформер из библиотеки Hugging Face `transformers`.

*   **Название модели:** `blanchefort/rubert-base-cased-sentiment`
*   **Описание решаемой задачи:** **Классификация текста**. Модель обучена определять эмоциональную окраску русскоязычных текстов и относить их к одному из трех классов:
    *   `positive` (позитивная)
    *   `negative` (негативная)
    *   `neutral` (нейтральная)

Это позволяет боту не просто обрабатывать запросы, но и понимать эмоции пользователей.

## 🛡️ Система антиспама и управления доступом

В бота встроена комплексная система защиты от спама и управления доступом:

*   **Белый список:** Доступ к боту могут иметь только пользователи, добавленные администратором.
*   **Черный список:** Возможность заблокировать нежелательных пользователей.
*   **Rate Limiting:** Ограничение на количество запросов от одного пользователя (5 в минуту, 30 в час, 100 в день).
*   **Административные команды:** Специальные команды `/whitelist`, `/blacklist` и `/stats` для управления ботом доступны только администратору (ID задается в коде).
